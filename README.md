# Mermaid Advanced

A modern, beautiful Mermaid.js editor and previewer built with React, Tailwind CSS, and Vite.

## Features

- **Live Preview**: Real-time rendering of Mermaid diagrams.
- **Custom Themes**:
  - Default
  - Industrial (Dark, Monospaced)
  - Hand Drawn (Sketchy style)
  - Studio Ghibli (Nature inspired)
  - Retro (Neon/Terminal style)
- **High Quality Export**:
  - PNG (Transparent background)
  - JPG (With background theme)
- **Modern UI**: Clean, responsive design.

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS v4
- Mermaid.js
- Vite
- Lucide React (Icons)
- html-to-image (Export)

## Getting Started

1. **Install dependencies**
   ```bash
   pnpm install
   ```

2. **Start Development Server**
   ```bash
   pnpm dev
   ```

3. **Build for Production**
   ```bash
   pnpm build
   ```
